# notlarim.net

**notlarim.net**; Web tabanlı, kullanıcıya özel bir sanal not defteri uygulamasıdır. Uygulama vasıtasıyla kullanıcının cihazdan bağımsız şekilde notlarına erişerek değişiklik yapabilmesi amaçlanmaktadır.
## ⚙️ Özellikler 

- Kullanıcı Sayfası
- Yönetici Paneli

## 🛠 Araçlar
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- CodeIgniter 4
